# PomodoroPet Desktop

Arcade-style Pomodoro focus timer desktop app.

## Install from npm

```sh
npm install -g pomodoropet-desktop
```

Then launch it with:

```sh
pomodoropet
```

## Development

```sh
npm install
npm run dev
```

## Publish

```sh
npm login
npm publish
```

The package runs `npm run build` before packing so the published app includes the latest Vite build.
